# 更新日志

所有重要的变更都将记录在此文件中。

## [3.0.0] - 2024-03-18

### 🎉 正式发布

这是大圣 VS Code 扩展的首个正式发布版本，代号为 "**六耳猕猴**"。

### ✨ 新增功能

#### 🌐 语言通 - YAML 翻译
- 支持 YAML 多语言文件智能同步翻译
- 自动检测新增/删除的 key
- 支持 50+ 种目标语言
- 保持原有数据结构

#### 📝 译真经 - Markdown 翻译
- 智能保护 frontmatter 元数据
- 保护代码块、链接、图片不被翻译
- 支持批量翻译
- 自动翻译 frontmatter 中的 title/description

#### 🎨 画皮术 - 封面生成
- 5 种精美风格：科技、简约、渐变、插画、商务
- 多种比例可选：16:9、1:1、9:16、4:3、21:9
- 从 Markdown 文件自动提取标题和作者
- 生成 SVG 格式封面

#### ✍️ 妙笔生花 - 博客生成
- 5 种写作风格：版本发布、教程、新闻、深度分析、故事
- 支持从 Release Notes 文件生成
- 支持手动输入主题和要点
- 自动生成 frontmatter

### 🖥️ 用户界面

- **侧边栏视图** - 快速访问所有功能
- **Dashboard 面板** - 可视化功能入口
- **右键菜单集成** - 文件右键快捷操作
- **状态栏按钮** - 一键打开面板
- **快捷键支持** - `Ctrl+Shift+D` 打开面板，`Ctrl+Shift+T` 翻译选中

### ⚙️ 配置系统

- 完整的 VS Code 配置集成
- 支持环境变量存储 API Key
- 可自定义默认语言和风格
- 自动检测 YAML 文件变化

### 🔧 技术特性

- TypeScript 全类型支持
- Webview 现代化界面
- 进度通知和取消支持
- 完善的错误处理
- 日志输出通道
