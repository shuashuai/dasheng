# Release Notes - Version 1.5.0

**Release Date:** 2026-03-17

## 🎉 New Features

- **AI-Powered Translation**: Automatically translate YAML and Markdown files with context-aware AI
- **Multi-Language Support**: Support for 20+ languages including Chinese, Japanese, Korean
- **Smart Content Protection**: Code blocks, links, and frontmatter are preserved during translation
- **Batch Processing**: Translate multiple files in one command

## 🚀 Improvements

- Enhanced CLI interface with colorful output and progress indicators
- Improved error handling and user-friendly error messages
- Added support for custom AI providers (OpenAI, Kimi, DeepSeek)
- Optimized translation speed with parallel processing

## 🐛 Bug Fixes

- Fixed SSL connection issues with certain API providers
- Resolved encoding problems with non-ASCII characters
- Fixed memory leak during batch translation of large files
- Corrected handling of nested YAML structures

## ⚠️ Breaking Changes

- CLI command structure has been refactored, please update your scripts
- Configuration file format changed from JSON to YAML

## 📝 Documentation

- Updated README with new examples
- Added comprehensive API documentation
- Created video tutorials for common use cases
